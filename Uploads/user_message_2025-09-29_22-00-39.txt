KEY IMPLEMENTATIONS
1) chat_client.py
- HttpModel(base_url, api_key, timeout). POST to /v1/chat/completions or /v1/chat. Tenacity retries, exponential backoff. Record token usage in logs.
- TieredChatClient(small, medium, large). Iterate tiers; circuit-breaker per tier.

2) self_loop.py
- __init__(chat_client, rag_search=None, critic_client=None).
- self_loop(q, path, rounds=3, tau=0.7):
  - Planner (bullets).
  - Draft via run_path(path, draft_prompt) using LFM2 (chat_client).
  - Verifiers:
    - CODE: run code_verifier tests; enforce timeout and memory caps.
    - MATH: symbolic and numeric checks.
    - RAG/GENERAL: verify bracketed citations [1], provenance score from retrieval_verifier.
  - Critique with critic_client (DeepSeek V3). If None, fallback to chat_client.
  - Final revision with LFM2 using critique + verifier findings. Stop early if confidence ≥ τ; else one debate pass.

3) service/api.py
- FastAPI with CORS allowlist from env.
- Authenticator.verify() against AUTH_TOKENS.
- RateLimiter.allow(user, route, limit, window) with in-proc counter (pluggable to Redis later).
- Endpoints: GET /healthz, GET /readyz, POST /ask {question, vision?, images?}.
- Metrics via observability.record_request(route, latency), record_error(route, kind).

4) service/bootstrap.py
- Build TieredChatClient from env:
  - SMALL_MODEL_URL, MEDIUM_MODEL_URL, LARGE_MODEL_URL (+ *_KEY).
- Build DeepSeek critic HttpModel from env: DEEPSEEK_URL, DEEPSEEK_API_KEY.
- Build retriever = QdrantESRetriever(QDRANT_URL, ES_URL).
- Assemble SelfLoop(chat_client=tiers, critic_client=deepseek, rag_search=retriever.search).
- create_app(auth, rate_limiter, allowed_origins, loop_handler).

5) retrieval/qdrant_es.py
- SentenceTransformer(“intfloat/e5-small”) for dense.
- Qdrant REST for vectors; Elasticsearch BM25 for sparse.
- search(q,k): dense(50) + sparse(50) → merge → rerank with cross-encoder “BAAI/bge-reranker-base” → top-k docs.

6) verifiers/code_verifier.py
- Extract python fences. Run with subprocess + RLIMIT_CPU=2s, RLIMIT_AS=256MB, FS cap, timeout=3s. No network.
